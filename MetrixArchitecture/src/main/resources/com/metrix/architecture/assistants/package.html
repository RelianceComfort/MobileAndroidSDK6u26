The Assistants package contains classes designed to make common functions that you do while implementing layouts easier. The MetrixControlAssistant class contains methods to get and set layout values, get a reference to layouts and to populate spinners. The MetrixDialogAssistant class contains methods to display popup dialogs as alerts, user confirmation requests, etc. The MetrixLocationAssistant class contains methods for easy interaction with the GPS radio on the device, while the MetrixMessageAssistant contains methods for interacting for data exchange messages that may or may not exist. Finally the MetrixPageAssistant contains methods for interacting with the screen layout itself.
